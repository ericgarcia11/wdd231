Site Name:

Trip Planning Website

Site Purpose:

A website that helps users plan their trips easily by providing popular trip ideas and allowing users to enter their own trip details to find ticket options. The site will fetch data from a free API and email the results to users.

Scenarios:

How can I find cheap tickets to my trip?

Where can I trip next summer?

Color Schema:

1th Color: #13293D. Will be used in the head, nav background and footer.

2th Color: #E8F1F2. Will be used as the text color where the primary color is used as background color.

3th Color: #1B98E0. Will be used in hover efects, like the nav hover where the 1th color is used as the background color.

4th Color: White. Used as the main background.

Any variation of this ones is transparent efects.

Typography:

The only font will be "Inter", sans-serif. Will be used in the entire site. The same used in this site plan page.
